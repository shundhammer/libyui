Detailed Description

template<typename T>
class YBothDim< T >

Template class for two-dimensional entities, such as

  • width, height
  • x_pos, y_pos
  • hStretchable, vStretchable

Precondition: type T needs to have a default constructor (which all simple types like int, long, bool have).
